# ApexPlanet-Internship-Task3
# Web Application Security Testing (DVWA)

## Internship Task 3

This project demonstrates exploitation and mitigation of common web vulnerabilities using DVWA.

---

## Tools Used

* Kali Linux
* DVWA
* Burp Suite
* Apache Server
* MySQL

---

## Vulnerabilities Demonstrated

### SQL Injection

Extracted database information using SQL injection payloads.

### Cross-Site Scripting (XSS)

Demonstrated stored and reflected XSS attacks.

### Cross-Site Request Forgery (CSRF)

Created malicious form to change user password.

### File Inclusion

Demonstrated Local File Inclusion attack.

### Burp Suite

Intercepted and modified HTTP requests.
